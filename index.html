Portfolio

Summary Description:

A simulation using evolutionary algorithms and autonomous steering. The agents spawn with a unique genetic code that contains different values which effect their behavior in the environment. They have a value for desire for food and poison. They also have sight values for recognizing the food and poison in the environment within a given range. They move in straight lines and the desire for food or poison modifies their vectors towards or away the target once it is within their sight radius. Each vehicle also has different min/max speed values in their genetic code. There is also a chance they will clone themselves and retain the same dna values. There is mutation which happens rarely and slightly modifies the genetic values of the agent at cloning. More healthy agents have twice the chance to clone themselves.
